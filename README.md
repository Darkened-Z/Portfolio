# Zeeshan Khan — Portfolio

Freelance software engineer. Solo builder shipping working POS, booking, ERP and WhatsApp bot systems for small businesses. Every product on the site is a live interactive demo.

- **Live:** [devora-psi.vercel.app](https://devora-psi.vercel.app) *(pending rename)*
- **Contact:** 7eeshan.ahmad@gmail.com · [LinkedIn](https://www.linkedin.com/in/zeeshan-khan)

## Stack

Next.js 16 (App Router, Turbopack) · React 19 · TypeScript · Tailwind v4

## Local dev

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Routes

- `/` — landing (hero, work, testimonials, insights, competencies, contact)
- `/pos` — Point of Sale demo
- `/money-tracker` — Money tracker demo
- `/order-bot` — WhatsApp order bot demo
- `/booking` — Booking system demo
- `/erp` — ERP / business manager demo

## Design system

See `src/app/globals.css` for the token layer. Magazine-Crest direction — warm-black background, cream text, gold highlighter accent, heavy slab display (Roboto Slab 900) + Playfair Display crest + Manrope body + JetBrains Mono meta labels.
